# React-To-Do-App

To run this app you should create a **.env** file in todo_app folder.
**.env** file should contain Firebase configuration  like below.
___________________________________________

REACT_APP_APIKEY=your_apikey
REACT_APP_AUTHDOMAIN=your_authdomain
REACT_APP_DATABASEURL=
REACT_APP_PROJECTID=
REACT_APP_STROAGEBUCKET=
REACT_APP_MESSAGINGSENDERID=
REACT_APP_APPID=
___________________________________________

After .env file is created you should first run **npm install** and then you can run **npm start**.
